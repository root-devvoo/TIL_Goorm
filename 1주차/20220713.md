###### 서브넷 마스크 / prefix

- 같은 대역임을 알려주는 역할
- 같은 대역으로 지정하고 싶다면 대표적으로

**서브넷 마스크** : 255.255.255.0 으로 지정할 수 있음

**프리픽스** : ip 주소 뒤에 /24로 표기할 수 있음

- 프리픽스의 숫자가 1 증가하면 ip 대역이 2분할됨

**게이트웨이** : 관문, 패킷이 나가고 들어오는 일종의 문 역할
